# my-awesome-turbowarp-plugins

TurboWarp 自定义扩展插件合集。

## 插件列表

### 摄像头截图 (Camera Capture)

从 TurboWarp 摄像头中捕获当前帧并保存为 PNG 图片。

#### 功能

- **捕获当前帧** — 从视频设备抓取当前画面
- **保存帧为 [文件名]** — 将捕获的画面下载为 PNG 文件（默认文件名 `capture.png`）

#### 使用方法

1. 打开 [TurboWarp](https://turbowarp.org/)
2. 确保项目中已启用摄像头（使用 Scratch 内置的「侦测」类别中的视频相关积木）
3. 加载自定义扩展，粘贴 `camera_capture.js` 的 URL 或内容
4. 在积木面板中找到「摄像头截图」类别，使用对应积木

#### 使用流程

```
先执行「捕获当前帧」→ 再执行「保存帧为 capture.png」
```

> 注意：必须先捕获帧，再保存。保存前请确保 TurboWarp 的视频设备已开启。

### 百炼大模型 (DashScope AI)

调用阿里百炼大模型 API（通义千问），支持文字对话、摄像头画面保存、图片理解、语音识别（STT）、语音合成（TTS）。

#### 积木

**文字对话：**

| 积木 | 类型 | 说明 |
|------|------|------|
| 使用 API Key [KEY] 向模型 [MODEL] 提问 [QUESTION] | 指令 | 发送问题到百炼 API，异步请求 |
| 当收到大模型回复时 | 事件 | API 返回结果时触发 |
| 大模型的回答 | 报告 | 获取最近一次大模型的回答内容 |

**摄像头 / 图片理解：**

| 积木 | 类型 | 说明 |
|------|------|------|
| 保存摄像头画面为 [FILENAME] | 指令 | 将当前摄像头画面保存为图片，并缓存这一帧供后续发送 |
| 使用 API Key [KEY] 向视觉模型 [MODEL] 发送摄像头画面并提问 [PROMPT] | 指令 | 捕获当前摄像头画面并立即发送给视觉模型进行理解 |
| 使用 API Key [KEY] 向视觉模型 [MODEL] 发送最近一次保存的图片并提问 [PROMPT] | 指令 | 发送最近一次保存或缓存的摄像头图片给视觉模型 |
| 当图片理解完成时 | 事件 | 图片理解返回结果时触发 |
| 图片理解结果 | 报告 | 获取最近一次图片理解结果 |

**语音识别（STT）：**

| 积木 | 类型 | 说明 |
|------|------|------|
| 开始录音 | 指令 | 请求麦克风权限并开始录音 |
| 停止录音并使用 API Key [KEY] 识别语音 | 指令 | 停止录音，调用 Qwen ASR 语音识别 |
| 当语音识别完成时 | 事件 | 识别完成时触发 |
| 语音识别结果 | 报告 | 获取最近一次语音识别的文字 |

**语音合成（TTS）：**

| 积木 | 类型 | 说明 |
|------|------|------|
| 使用 API Key [KEY] 将 [TEXT] 合成为语音 | 指令 | 调用 CosyVoice 将文字转语音 |
| 当语音合成完成时 | 事件 | 语音合成完成时触发 |
| 播放合成语音 | 指令 | 播放最近一次合成的语音 |
| 停止播放语音 | 指令 | 停止当前播放的语音 |

#### 使用方法

1. 在 [阿里云百炼平台](https://bailian.console.aliyun.com/) 获取 API Key
2. 加载扩展，在提问积木中直接传入 API Key

#### 使用流程

**文字对话：**
```
当 🟢 收到大模型回复时
  说（大模型的回答）

使用 API Key [sk-xxx] 向模型 [qwen-plus] 提问 [你是谁？]
```

**语音问答（STT → LLM → TTS）：**
```
当 🟢 语音识别完成时
  使用 API Key [sk-xxx] 向模型 [qwen-plus] 提问（语音识别结果）

当 🟢 收到大模型回复时
  使用 API Key [sk-xxx] 将（大模型的回答）合成为语音

当 🟢 语音合成完成时
  播放合成语音

开始录音
  ↓ （等待用户说话）
停止录音并使用 API Key [sk-xxx] 识别语音
```

**摄像头图片理解：**
```
当 🟢 图片理解完成时
  说（图片理解结果）

使用 API Key [sk-xxx] 向视觉模型 [qwen3.6-flash] 发送摄像头画面并提问 [请描述当前摄像头画面]
```

**先保存，再发送同一张图：**
```
保存摄像头画面为 [capture.png]
使用 API Key [sk-xxx] 向视觉模型 [qwen3.6-flash] 发送最近一次保存的图片并提问 [请分析刚才保存的图片]
```

> 注意：扩展需要以非沙箱模式运行。使用摄像头相关积木前，请先在 TurboWarp 中开启视频设备。下载到本地的图片文件不会被浏览器重新读回扩展，因此“发送最近一次保存的图片”实际使用的是扩展在保存时同步缓存的那一帧图片。语音识别默认使用 `qwen3-asr-flash` 模型（适合短音频，建议控制在 5 分钟内），图片理解默认使用 `qwen3.6-flash` 模型，语音合成使用 `cosyvoice-v3.5-flash` 模型。

## 如何在 TurboWarp 中加载自定义扩展

1. 将 `.js` 文件托管到可公开访问的 URL（如 GitHub Pages、jsDelivr 等）
2. 在 TurboWarp 编辑器中，点击左下角「添加扩展」
3. 选择「自定义扩展」，输入文件 URL 并确认加载

也可以直接将文件内容复制到 TurboWarp 的自定义扩展加载器中。
